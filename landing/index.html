Security architecture

Most extensions see everything.
Ours sees only what it needs.

In 2025, 300+ malicious AI extensions were caught harvesting credentials and browsing history. AetherTab was built as the direct response.

API key storageCommon: plaintext in chrome.storageAetherTab: AES-GCM 256-bit encrypted. PBKDF2, 100k iterations, unique salt per install.
Host permissionsCommon: <all_urls> broad accessAetherTab: Explicit allowlist — 5 domains only. No wildcard.
AI output renderingCommon: raw innerHTML (XSS vector)AetherTab: textContent only. No innerHTML from AI responses. Ever.
Content Security PolicyCommon: missing or weakAetherTab: Strict script-src 'self'. No eval. No inline scripts.
Analytics / telemetryCommon: Google Analytics, Mixpanel, etc.AetherTab: None. Zero. Not a tracker in sight.

storage · alarms · sidePanel
